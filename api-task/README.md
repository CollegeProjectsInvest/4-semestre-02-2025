# API TASK

## Configurar projeto:

1. Instalar dependências

```bash
npm install
```

2. Configurar variáveis de ambiente

renomeia o arquivo **.env.example** para **.env**

3. Configurar banco de dados:

```bash
npm prisma -- generate
```

4. Abrir Prisma Studio (opcional)

```bash
npm run prisma -- studio
```

5. Rodar API:

```bash
npm start
```
